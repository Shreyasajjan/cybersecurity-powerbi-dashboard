🔐 Cybersecurity Threat Monitoring Dashboard (Power BI)
📊 Project Overview

The **Cybersecurity Threat Monitoring Dashboard** is a data visualization project built using **Power BI**.
It analyzes cybersecurity attack data and presents insights through interactive dashboards to help monitor and understand cyber threats.

This project demonstrates how **data analytics can support cybersecurity teams** in identifying attack patterns, affected systems, and potential risks.

---

🎯 Project Objectives

* Analyze cybersecurity attack data.
* Identify the most common **attack types**.
* Understand **target systems and vulnerabilities**.
* Visualize the **impact of cyber attacks**.
* Present insights using an **interactive Power BI dashboard**.

 🛠️ Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **CSV Dataset** – Cybersecurity attack dataset
* **Data Analysis** – Attack pattern exploration
* **GitHub** – Project version control and documentation

📈 Dashboard Features

The Power BI dashboard includes the following visualizations:

* **Total Cyber Attacks Overview**
* **Attack Type Distribution**
* **Security Category Analysis**
* **Target System Analysis**
* **Impact of Cyber Attacks**
* **Detection Methods & Mitigation Strategies**

These visualizations help in understanding cybersecurity threats more effectively.


📊 Dataset

The dataset used in this project contains cybersecurity attack scenarios including:

* Attack Type
* Category
* Target Type
* Vulnerability
* Impact
* Detection Method
* Security Solutions

Dataset Source: Kaggle Cybersecurity Dataset

🚀 How to Use This Project

1. Download the dataset from the **dataset folder**.
2. Open the **Power BI dashboard file (.pbix)**.
3. Explore the interactive dashboard.
4. Use filters to analyze specific cyber attack scenarios.
